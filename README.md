# CCTransfer

All California Community College Transfer Resources in One Place

## About
CCTransfer (cacctransfer.com) is a comprehensive website providing California Community College students with transfer information and resources for UC and CSU systems. The focus is to consolidate all relevant transfer information in one accessible location.

## Structure
- **index.html** - Landing page with hero section and highlights
- **transfer-programs.html** - TAG, TAP, H2H, CCCP Scholars, Transfer Pathways+
- **opportunities.html** - Clubs, research, internships, professional development
- **financial-aid.html** - FAFSA, grants, scholarships, financial aid calendar
- **honors.html** - Honors programs and requirements by college
- **deadlines.html** - Comprehensive deadline calendar with filtering
- **resources.html** - ASSIST tutorials, application guides, official links
- **campuses.html** - CC directory and location finder
- **contact.html** - Contact form and team information

## Development
This is a static website designed to be easily maintainable and updatable by transfer students and community members.

## Contributing
Transfer students are invited to join the project and contribute to keeping information current and comprehensive.
